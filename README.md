# Trouble Board Game

**Run Tests**

```
rake
```

or

```
bundle exec rspec
```

----

**Gem Installs**

Add new gem in `Gemfile` and run:

```
bundle install
```

----

**Adhoc Progamming and Debuging**

Enter an IRB/console with the following command.  All libraries are loaded in
the console.

```
rake console
```

or

```
rake c
```
